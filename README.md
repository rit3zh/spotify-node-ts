## @rit3zh/spotify.ts

Spotify TypeScript wrapper.

## Installation

```bash
yarn add @rit3zh/spotify.ts
```

## Usage

```ts
import * as Spotify from "@rit3zh/spotify.ts"; // importing everything
```
